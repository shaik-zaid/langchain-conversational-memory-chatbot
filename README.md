# LangChain Conversational Memory Chatbot

This project demonstrates how to build a conversational AI system with memory using LangChain.

It focuses on understanding how chatbots can retain context across multiple interactions using session-based message history.

---

## 🚀 Features

- Conversational chatbot with memory
- Session-based chat history handling
- Message trimming for token control
- LCEL (LangChain Expression Language) pipelines
- Step-by-step learning via notebooks
